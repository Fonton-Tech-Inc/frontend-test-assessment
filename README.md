# Todo App - Frontend Assessment

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Assessment Questions

### 1. What improvements would you suggest from a UI/UX perspective for this app?

Consider factors such as usability, design consistency, accessibility, and overall user experience.

### 2. How would you improve the codebase to enhance its quality?

Think about aspects like readability, maintainability, performance, and scalability.
